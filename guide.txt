# This is guide on how to setup dotfiles. 
# Was done on Ubuntu.
# It may not work for your distro.

1. Run 'sudo apt-get update' command followed by 'sudo apt-get upgrade'.
2. Create symlinks to bashrc (~/.bashrc) and bash_aliases (~/.bash_aliases).
	Example: ln -s ~/Downloads/dotfiles/bashrc ~/.bashrc
3. Install tmux using 'sudo apt-get install tmux' command.
4. Create symlink to tmux.conf (~/.tmux.conf). 
5. Create 'tmux_init' profile in terminal (i am using Gnome Terminal).
6. Add tmux_login.sh script to your startup scripts. 
7. Install VIM using 'sudo apt-get install' command.
8. Create symlink-s to vimrc (~/.vimrc) and vim (~/.vim).
